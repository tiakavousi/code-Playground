## 1. C program that takes a number as input: 
```
#include <stdio.h>

int main() {
    int number;

    // Ask the user for input
    printf("Enter a number: ");
    
    // Read the input number
    scanf("%d", &number);
    
    // Print the entered number
    printf("You entered: %d\n", number);

    return 0;
}
```
